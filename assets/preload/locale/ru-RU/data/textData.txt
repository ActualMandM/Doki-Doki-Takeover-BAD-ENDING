// DESCRIPTION
descAddOffset::Добавляет задержку к нотам, применяется ко всем песням.
descKeyBindings::Сменить назначения клавиш для каждой ноты.
descDownscroll::Меняет направление появляющихся нот (Downscroll - сверху, Upscroll - снизу)
descGhostTap::Ghost Tapping - когда вы нажимаете на ноту и вам не засчитывается промах.
descJudgement::Изменить кол-во безопасных кадров. 
descCurJudgement::Текущее кол-во безопасных кадров.
descFPSCap::Поставить лимит на количество кадров в секунду. (Влево - 10, Вправо +10. Shift чтобы ускориться)
descCurFPSCap::Текущий лимит кадров в секунду.
descRefreshRate::Частота обновления.
descScroll::Изменить скорость прокрутки нот. (Влево -0.1, Вправо +0.1. 1 - скорость по умолчанию)
descCurScroll::Текущая скорость прокрутки.
descAccuracyMode::Режимы: Точный = Простой метод, Сложный = Основывается на милисекундах
descReset::Включает возможность нажать на R чтобы получить гейм овер.
descCustomize::Меняйте положение элементов интерфейса сколько душе угодно.
descCustomizeState::Перетаскивайте счёт, чтобы изменить его позицию. R чтобы вернуть его на место. Q/E - приблизить/отдалить. Escape - выход.
descDistract::Включает отвлекающие элементы, влияющие на игровой процесс.
descFPSRainbow::Счетчик кадров в секунду будет мигать, как радуга. Красивенько!
descAccuracy::Показывает информацию о точности.
descNPS::Показывает количество НВС (Нот в секунду)
descPosition::Показывает текущий прогресс песни. (Ввиде шкалы)
descFPSCount::Включить счетчик кадров в секунду.
descReplay::Показывает ваши прошлые игры.
descFlashing::Переключатель мерцающих цветов, которые могут вызвать эпилепсию или судороги. 
descWatermark::Показывает водяные знаки Kade Engine.
descBotplay::Если эта опция включена, за вас будет играть бот. 
descGFCountdown::Если эта опция включена, Девушка будет отсчитывать "3, 2, 1" перед песней.
descKeyBindKBCurrent::Текущий ввод: Клавиатура.
descKeyBindGPCurrent::Текущий ввод: Джойстик.
descKeyBindTAB::Нажмите TAB чтобы переключить.
descKeyBindKBControls::Escape чтобы сохранить настройки, Backspace чтобы выйти без сохранения. Enter чтобы сменить назначение клавиши.
descKeyBindGPControls::Правый триггер чтобы сохранить настройки, Левый Триггер чтобы выйти без сохранения. Кнопка "Старт" чтобы сменить назначение клавиши.
descKeyBindBlacklist::в черном списке!
descCharacterCache::Уменьшает время загрузки при старте игры, кэшируя персонажей. 
descSongCache::Уменьшает время загрузки при старте игры, кэшируя каждую песню. 
descMusicCache::Уменьшает время загрузки при старте игры, кэшируя музыку в меню и т.д. 
descSoundCache::Уменьшает время загрузки при старте игры, кэшируя каждый звуковой эффект. 
descCaching::Вы можете уменьшить время загрузки игры, переключая опции в этом меню.
descDisableCaching::(нажмите D чтобы отключить при следующей загрузке)
descMiddleScroll::Ставит линию прокрутки нот в середину.
descLaneUnderway::Включает фон для линии прокрутки нот.
descLaneUnderwayControl::Текущий уровень прозрачности фона для линии прокрутки нот.
descSaveReset::Полностью обнуляет ваш файл сохранения.
descDialogueSkip::Нажмите ESCAPE чтобы пропустить диалог.
descGameJolt::Войдите в свой аккаунт Game Jolt чтобы получать достижения!
descMirrorMode::Поменяйтесь местами и играйте за оппонента!
descHitSound::Проигрывает звук когда вы попадаете по ноте.
descHitSoundControl::Текущий уровень громкости звуков попадания
descNoteSplash::Adds a splash to any sick notes that you hit!
descMillisecond::Shows the timing of the note that you hit in milliseconds.
